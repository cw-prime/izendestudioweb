How to Implement Dark Mode on a PHP Website

Dark mode reduces eye strain in low-light environments and is now expected by many users. Here’s a pragmatic approach to add a lightweight, accessible dark mode to a PHP website.

1. Use CSS variables

Define a palette with custom properties in your main stylesheet and provide a dark-mode override by applying a class to the <body> element.

:root {
  --background: #fff;
  --text: #222;
}

body.dark-mode {
  --background: #181818;
  --text: #eee;
}

2. Toggle with JavaScript

Add a toggle control in your header and persist the preference in localStorage. On page load, apply the stored class to body so the theme persists across navigation.

3. (Optional) Persist server-side

If your site has user accounts, save the preference in the user profile and render a server-side class via a PHP include so the theme is applied before the page loads.

Using CSS variables keeps your implementation flexible and low-cost. Test contrast and keyboard accessibility before shipping.
